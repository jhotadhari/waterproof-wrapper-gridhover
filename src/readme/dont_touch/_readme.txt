A grid layout wrapper. Uses pure CSS3 effects on hover.!


== Description ==
You need the \"Waterproof Wrap Query\" Plugin to use this Plugin.
Wrapper to extend the 'Waterproof Wrap Query' Plugin.
A grid layout. Uses pure CSS3 effects on hover.
Inspired by that [Codrops Article](https://tympanus.net/codrops/2011/11/02/original-hover-effects-with-css3/)
The First 5 styles are supported currently.


## How to use:
* Its basicly documented in the \"Waterproof [wrap_query] shortcode docs\" metabox on the post edit sceen.

## Requirements:
* php 5.6
* Any Waterproof wrapper

Thanks to CMB2 for their easy metaboxes and optionpages!

== Installation ==
Requirements:
* php 5.6
* Waterproof Wrap Query Plugin

Upload and install this Plugin in the same way you\'d install any other plugin.
